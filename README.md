# Zev Portfolio

Personal portfolio for AI Product & Data Analyst applications.

## Run Locally

```bash
npm run dev
```

## Check

```bash
npm run check
```

## Environment Variables

Create `.env.local` locally and configure the same variables in Vercel:

```text
XIAOMI_API_KEY=your_api_key_here
XIAOMI_BASE_URL=https://token-plan-sgp.xiaomimimo.com/v1
XIAOMI_MODEL=MiMo-V2.5-Pro
```

Do not commit real API keys.
